# Personal website

Minimal single-page site for [Andrei Constantinescu](https://github.com/andreiconst). One `index.html` file — no build step.

## Preview locally

```bash
open index.html
# or
python3 -m http.server 8000
```

Then open http://localhost:8000

## Stack

- HTML + [Tailwind CSS](https://tailwindcss.com) (CDN)
- [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- System light/dark mode (`prefers-color-scheme`)
